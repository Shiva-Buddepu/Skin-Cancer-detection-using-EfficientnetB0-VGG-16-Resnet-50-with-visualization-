# Skin Cancer Detection using EfficientNetB0, VGG-16 and ResNet-50

This project focuses on detecting skin cancer from dermoscopic images using Deep Learning models.  
Multiple pretrained architectures are implemented and compared:

- EfficientNetB0
- VGG-16
- ResNet-50

The goal is to build an accurate classification model for early detection of skin cancer.

---

##  Project Overview

Skin cancer detection is a critical medical imaging task.  
This project:

- Performs image preprocessing and augmentation
- Trains and fine-tunes multiple pretrained CNN models
- Handles class imbalance using data augmentation
- Evaluates and compares model performance
- Visualizes dataset distribution and results

---


##  Models Used

### 1️⃣ EfficientNetB0
- Lightweight and highly efficient architecture
- Fine-tuned for skin cancer classification

### 2️⃣ VGG-16
- Deep CNN with sequential convolution layers
- Transfer learning applied

### 3️⃣ ResNet-50
- Residual Network with skip connections
- Helps in training deeper architectures

---



##  Workflow

1. Load and preprocess dataset
2. Perform data augmentation (rotation, flipping, scaling)
3. Split into training and validation sets
4. Apply transfer learning using pretrained models
5. Train and fine-tune models
6. Evaluate using accuracy and loss metrics
7. Compare model performance

---

##  Evaluation Metrics

- Accuracy
- Training & Validation Loss
- Model Comparison
